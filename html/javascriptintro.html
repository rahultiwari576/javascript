Integer

An integer is a whole number (not a fraction) that can be positive, negative, or zero. Integers are a commonly used data type in computer programming. ... For example, whenever a number is being incremented, such as within a "for loop" or "while loop," an integer is used.

Variable

In computer programming, a variable or scalar is a storage location (identified by a memory address) paired with an associated symbolic name, which contains some known or unknown quantity of information referred to as a value.
